Karing 科学上网工具使用教程

2026 最新版完整指南｜适用于 iOS / Android / Windows / macOS

一、Karing 是什么？

Karing 是一款跨平台网络代理工具，基于 sing-box 内核开发， 兼容 Clash、V2ray/V2fly、Sing-box、Shadowsocks 等多种协议和订阅链接。

主要特点：

  • 多协议支持：支持 Shadowsocks、ShadowsocksR、VMess、VLESS、Trojan、Tuic、Socks、HTTP、Hysteria、Hysteria2、WireGuard 等主流协议。
  • 跨平台兼容：支持 Windows、macOS、iOS、Android、HarmonyOS 等多个平台。
  • 智能路由：一套路由规则可应用于多个订阅源，自动分流常见网站。
  • 高性能内核：魔改版 Sing-box 核心，性能和稳定性都比较强。
  • 新手友好：提供新手模式和默认规则组，安装完不用折腾就能直接用。

二、下载安装 Karing

1. iOS(iphone / ipad) / Mac 用户

  1. 准备一个非中国大陆的 Apple ID（美区 / 港区等）。
  2. 在 App Store 搜索 karing 或 karing vpn。
  3. 找到图标为蓝紫渐变的 Karing 应用后点击“获取”安装。
  4. 也可以通过 TestFlight 链接安装测试版。

iOS / macOS 相关地址：

  • App Store：https://apps.apple.com/us/app/karing/id6472431552
  • TestFlight：https://testflight.apple.com/join/RLU59OsJ

2. Windows / Android / Harmony 用户

  1. 访问 Github Releases：https://github.com/KaringX/karing/releases
  2. Windows：下载文件名类似 karing_xxx_windows_x64.exe 的安装包，双击安装。
  3. Android / Harmony：下载 karing_xxx_android_arm.apk，拷贝到手机后安装。
  4. 如果 GitHub 无法访问，可以使用站内提供的备用下载地址。
提示：Windows 安装被 Defender 拦截时，点“显示详细信息” → “仍要运行” 即可。

三、首次启动与基础设置（以 iOS 为例）

1. 接受隐私协议

  1. 首次打开 Karing，会弹出隐私说明页面。
  2. 滑到页面底部，点击 Accept & Continue 继续。

2. 选择语言与地区

  1. 在语言列表中选择 简体中文，操作更方便。
  2. 接着选择当前所在的国家与地区，否则可能导致分流错误。
  3. 新手模式默认开启，保持默认直接下一步 / 完成即可。

3. 主界面预览

完成向导后，就能看到 Karing 的主界面，大致分为：

  • 顶部：运行时长、连接状态、总流量、速度等信息。
  • 中部：当前配置、规则 / 全局、我的配置、添加配置、DNS、分流等按钮。
  • 底部：测速、快捷连接、自动选择等功能入口。

四、配置 Karing（添加机场订阅）

要实现科学上网，需要给 Karing 添加一个有效的代理节点订阅。推荐新手直接使用支持 Karing / sing-box 的机场订阅。

1. 在机场后台复制订阅地址

  1. 登录机场后台，进入“仪表盘 / 我的订阅”。
  2. 点击一键订阅，选择导入到 sing-box / Clash Meta 等。
  3. 选择“复制订阅地址”，得到订阅链接。

2. 在 Karing 中添加配置链接

  1. 回到 Karing 主界面，点击底部“添加配置”。
  2. 在“配置链接/内容”框中粘贴刚才复制的订阅地址。
  3. 备注处可以写上机场名称，例如 karing.cc，方便区分。
  4. 点击右上角对勾保存，弹出提示时选择“确定”。
  5. 在“我的配置”列表中可以看到刚添加好的订阅，记得点一下“更新”。

五、选择节点并开启代理

1. 选择服务器节点

  1. 在主界面点击底部的“自动选择 >”进入服务器列表。
  2. 在服务器界面点击闪电 ⚡ 图标可以测速。
  3. 根据延迟和线路类型，选择一个合适的节点，或者使用自动选择。

2. 打开 Karing 代理开关

  1. 回到主界面，点击底部中间的开关按钮。
  2. 第一次开启会弹出“允许添加 VPN 配置”的系统提示，选择“允许”。
  3. 开关变成绿色、界面开始有流量统计，说明代理已经生效。

3. 测试外网访问是否正常

  1. 打开 Safari 或浏览器访问 youtube.com、google.com 等网站。
  2. 如果页面能正常加载，说明配置成功。

六、常见问题简要整理

1. Karing 支持哪些订阅类型？

Karing 支持 Clash、Sing-box、Shadowsocks、V2ray 等常见订阅类型，一般机场都可以直接兼容。

2. 添加 / 更新配置失败怎么办？

  • 先在浏览器中打开订阅链接，确认链接本身是否可用。
  • 如果打不开，可能是链接过期或域名被封，去机场后台复制新的订阅链接。
  • 也可以用订阅转换工具（如 sub-web）转成 sing-box / Clash 兼容格式后再导入。

3. 小米手机无法安装 Karing？

  • Karing 有可能被 MIUI 标记为高风险应用。
  • 解决办法：先断网或打开飞行模式，并关闭“安全守护-增强防护”，再尝试安装。

4. Windows 端使用报错？

  • 优先用管理员权限运行 Karing。
  • 检查系统中是否还有其它翻墙软件在运行，全部退出后再试。
  • 仍旧有问题时，可以尝试更新到最新版本或重新安装。
